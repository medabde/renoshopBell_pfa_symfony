# RenoshopBell
Symfony project.
# Introduction :
A simple and functionnal e-commerce website (with admin side).
# Features :
 - List the products that exist in the database.
 - Login as client/admin/superAdmin.
 - Create an account
 - Make Orders, update them...
   
# Libraries/Tools used :
 - EasyAdmin
 - Sass
 - Xampp
 - MySQL
